Upload service for Roskomnadzor notifications
=============================================

docker-compose services and helpers
-----------------------------------

* gostca - Russian PKI with GOST cryptograpy (https://github.com/schors/gost-russian-ca + some historical certs)
* upload - upload service (golang)
* verify - script for stage1 upload verification
* tools - some helpers
* nginx - HTTPS terminator
* server.etc - some server configs (rsyslog, logrotate, cron + letsencrypt)
